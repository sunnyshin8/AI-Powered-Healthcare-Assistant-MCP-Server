# Healthcare MCP Server Frontend (Next.js)

## Features
- Animated, dynamic UI
- Real-time dashboard metrics
- API playground for live endpoint testing
- MCP tool integration
- Responsive and healthcare-themed design

## Setup
```bash
npm install
npm run dev
```
